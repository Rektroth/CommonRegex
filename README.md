## Phone
The format is `+ISO REG-LOC-NUMB`. The dashes are required. `+ISO` and `REG` are optional.

```
^((([+][1-9][0-9]?[0-9]?)?[ ])?[1-9][0-9]{2}[-])?[1-9][0-9]{2}[-][1-9][0-9]{3}$
```
